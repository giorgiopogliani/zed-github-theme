# Github Inspired Theme for Zed

Install Pkl:

```bash
brew install pkl
```

Build all themes:

```bash
make
```

Build one theme:

```bash
make light
make cream
make dark
make darker
```

Generated themes:

- `github-light.json` — `My Github Light v2`
- `github-cream.json` — `My Github Cream v2`
- `github-dark.json` — `My Github Dark v2`
- `github-darker.json` — `My Github Darker v2`
